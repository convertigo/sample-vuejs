# Convertigo sample for Vue.js 

> A Vue.js project with the c8osdkvuejs and vue 2.0, vue-router and vuex starter kit for server side rendering.


## Build Setup

Requires Node.js v6+

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```
